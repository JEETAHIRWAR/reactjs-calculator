## Description

This is a simple calculator application built using ReactJS. It provides basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)


## Demo

You can try out the calculator app by visiting the [live demo](https://jeetahirwar.github.io/Calculator1/).

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

- Node.js: You need to have Node.js installed on your system to run this application. You can download it from [nodejs.org](https://nodejs.org/).

### Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/yourusername/reactjs-calculator-app.git


2. Change the directory to the project folder:

   ```bash
   cd reactjs-calculator-app


4. Install the dependencies:

   ```bash
   npm install


### Running the Application

1. Start the development server:

   ```bash
   npm start


2. Open your web browser and navigate to http://localhost:3000 to access the calculator app.

### Usage

- To perform arithmetic calculations, click the number buttons and operators.
- Click the "=" button to see the result.
- Click the "C" button to clear the input and start over.
